# She Can Foundation Website

A simple and responsive NGO website built for the **She Can Foundation** using **React** and **Tailwind CSS**.

## Overview

This project is designed to promote the mission of She Can Foundation, an organization focused on empowering women through education, mentorship, and community support.

The website includes key sections that provide information about the foundation, its impact, and opportunities for volunteering.

# Live Link

https://shecanfo.netlify.app/

## Features

- Responsive design
- Modern and clean user interface
- Home section
- About section
- Impact statistics section
- Volunteer section
- Contact section
- Footer
- Smooth navigation
- Mobile-friendly layout

## Technologies Used

- Tailwind CSS
- JavaScript
